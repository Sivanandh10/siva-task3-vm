# Build a Todo App with Git

You are a developer building a real Todo application. Every feature you add gets tracked with Git, just like on a real dev team.

---

## Your workspace

| Tab | What it does |
|-----|-------------|
| **Terminal** | Run Git commands |
| **Code Editor** | Write and edit your app files |
| **Live App** | See your Todo app running live |
| **Git History** | Watch your commits appear in real time |

---

## The project

You will build a working Todo app step by step:

- **Page 2** - Create the app skeleton and make your first commit
- **Page 3** - Add todo functionality on a feature branch
- **Page 4** - Merge the feature into main
- **Page 5** - Add styling, resolve a conflict

Click **Next page** to start.
